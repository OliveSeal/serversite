Kva er ein Server?

Ein server er ein programvare eller maskinvare som tilbyr ein eller fleire tenester til andre datamaskiner (Klientar). Servere kan tilby ulike funksjoner, ofte kalla "tenester", for eksempel deling av data eller ressursar mellom fleire klientar. Ein server kan bli brukt av fleire klientar og ein enkelt klient kan bruke fleire servere.

Kva kan bli brukt som ein server og kva er RAID?

Alle datamaskiner kan teknisk sett bli brukt til ein server, men server maskiner bruker ofte dyrare og meir haldbare komponentar. De bruker det for å få betre ytelse og sikkerheit. Harddiskane til servere bruker ofte RAID for betre beskyttelse imot Datatapp og Diskfeil.

Typer Servere: Web Server

En webserver er dataprogramvare og underliggande maskinvare som aksepterer forespørsler via HTTP (nettverksprotokollen opprettet for å distribuere webinnhold) eller den sikre varianten HTTPS. En brukeragent, vanlegvis ein nettlesar eller Webcrawler, startar kommunikasjon ved å sende en forespørsel om en nettside eller annan ressurs ved å bruke HTTP, og serveren svarer med innhaldet i den ressursen eller en feilmelding.
